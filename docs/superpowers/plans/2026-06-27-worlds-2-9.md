# Worlds 2–9 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the port play all 9 worlds in the original's faithful linear order (clear every node in a world → advance to the next world's map + boards), with a dev shortcut to start on any world.

**Architecture:** Approach A (lazy reload, shell-owned). The per-world node graph is baked into `world_graphs.gen.cpp`; `App` owns the world *number* + a world-parameterized `WorldMap` and only *requests* a world load via `pending_world()`. The SDL shell owns disk I/O: when a load is pending it swaps a `WorldResources` bundle (`D{n}` + `MONDE{n}.VEC`) and calls `App::enter_world`. The reload lands under the edge-to-centre darken.

**Tech Stack:** C++20, SDL3, Catch2, CMake (preset `windows-debug`), Python 3 for the RE extractor.

## Global Constraints

- **Build:** `cmake --build --preset windows-debug` (configure once with `cmake --preset windows-debug` if `build/` is absent).
- **Run tests (from repo root):** `& build/windows-debug/Debug/bumpy_tests.exe` for all, or `& build/windows-debug/Debug/bumpy_tests.exe "<exact test name>"` for one. Tests read the original files by bare name, so the working directory MUST be the repo root.
- **Never modify the original game files** (`D*.PAV/DEC/BUM`, `MONDE*.VEC`, `*.EXE`, `BUMSPJEU.BIN`, `DDFNT2.CAR`, …). Verify clean with `python tools/assets/manifest.py verify` before each commit.
- **Generated `*.gen.cpp` are committed source** under `src/` (same as `move_scripts.gen.cpp`, `object_anim.gen.cpp`). The extractor reads `analysis/generated/BUMPY.UNPACKED.EXE` (an ignored artifact; regenerate with `python tools/re/unpack_bumpy.py` if missing).
- **Data-segment addressing:** file offset = `0x11440 + DS_offset` in `BUMPY.UNPACKED.EXE`.
- **Per-world far-pointer tables (verified):** node graph at `DS:0x10c8`, positions at `DS:0x10ec`, each a 4-byte far pointer per world at `table + world*4` (world 1-based; index 0 is a junk slot). Graphs are 9-byte records `{state, up,up_d, down,down_d, left,left_d, right,right_d}` indexed by node (1-based, node 0 zero-padding), terminated by a record whose `state` byte is `0xff`. Positions are `(x,y)` little-endian word pairs indexed by `node-1`.

Create `tools/re/dump_world_graphs.py`:

```python
#!/usr/bin/env python3
"""Extract the per-world map node graphs + positions from BUMPY.UNPACKED.EXE and emit
src/game/world_graphs.gen.cpp. Far-pointer tables: graph DS:0x10c8, positions DS:0x10ec,
indexed by world (1-based; world W pointer at table + W*4). Graph records are 9 bytes
{state, up,up_d, down,down_d, left,left_d, right,right_d}, node 1-based (node 0 padding),
0xff state byte terminates. Positions are (x,y) LE words indexed by node-1.
See analysis/specs/screen-flow.md and docs/.../2026-06-27-worlds-2-9-design.md."""
import struct
import pathlib

ROOT = pathlib.Path(__file__).resolve().parents[2]
EXE = ROOT / "analysis/generated/BUMPY.UNPACKED.EXE"
OUT = ROOT / "src/game/world_graphs.gen.cpp"
DS_BASE = 0x11440
GRAPH_TBL = 0x10c8
POS_TBL = 0x10ec
WORLDS = 9

data = EXE.read_bytes()


def u16(off):
    return struct.unpack_from("<H", data, off)[0]


def fo(ds):
    return DS_BASE + ds


def ptr(table, world):
    return u16(fo(table) + world * 4)


def world_nodes(w):
    gbase = fo(ptr(GRAPH_TBL, w))
    pbase = fo(ptr(POS_TBL, w))
    nodes = []
    n = 1
    while data[gbase + n * 9] != 0xFF:
        rec = data[gbase + n * 9 : gbase + n * 9 + 9]
        up, dn, lf, rt = rec[1], rec[3], rec[5], rec[7]
        x = u16(pbase + (n - 1) * 4)
        y = u16(pbase + (n - 1) * 4 + 2)
        if not (0 <= x < 320 and 0 <= y < 200):
            raise SystemExit(f"world {w} node {n}: position out of bounds ({x},{y})")
        nodes.append((up, dn, lf, rt, x, y))
        n += 1
        if n > 64:
            raise SystemExit(f"world {w}: no 0xff terminator within 64 nodes")
    if not nodes:
        raise SystemExit(f"world {w}: empty graph")
    return nodes


worlds = [world_nodes(w) for w in range(1, WORLDS + 1)]

out = []
out.append("// GENERATED by tools/re/dump_world_graphs.py -- DO NOT EDIT BY HAND.")
out.append("// Per-world map node graphs + positions, baked from BUMPY.UNPACKED.EXE.")
out.append("// Graph far-ptr table DS:0x10c8, positions DS:0x10ec (world W at table+W*4).")
out.append("// Records {state,up,up_d,down,down_d,left,left_d,right,right_d}; 0xff terminates.")
out.append("// See analysis/specs/screen-flow.md and docs/superpowers/specs/2026-06-27-worlds-2-9-design.md.")
out.append('#include "game/world_graphs.h"')
out.append("")
out.append("#include <array>")
out.append("#include <cstddef>")
out.append("")
out.append("namespace bumpy {")
out.append("namespace {")
out.append("")
for i, nodes in enumerate(worlds, start=1):
    out.append(f"// World {i}: {len(nodes)} nodes. Fields {{up, down, left, right, x, y}}; [0] = sentinel.")
    out.append(f"constexpr std::array<MapNode, {len(nodes) + 1}> kWorld{i}{{{{")
    out.append("    {0, 0, 0, 0, 0, 0},")
    for idx, (up, dn, lf, rt, x, y) in enumerate(nodes, start=1):
        out.append(f"    {{{up}, {dn}, {lf}, {rt}, {x}, {y}}},  // node {idx}")
    out.append("}};")
    out.append("")
out.append("struct WorldTable {")
out.append("    const MapNode* nodes;")
out.append("    int count;")
out.append("};")
out.append("")
entries = ", ".join(f"{{kWorld{i}.data(), {len(worlds[i - 1])}}}" for i in range(1, WORLDS + 1))
out.append(f"constexpr std::array<WorldTable, {WORLDS}> kWorlds{{{{{entries}}}}};")
out.append("")
out.append("}  // namespace")
out.append("")
out.append("const MapNode& world_node(int world, int node) {")
out.append("    return kWorlds[static_cast<std::size_t>(world - 1)].nodes[node];")
out.append("}")
out.append("")
out.append("int world_node_count(int world) {")
out.append("    return kWorlds[static_cast<std::size_t>(world - 1)].count;")
out.append("}")
out.append("")
out.append("}  // namespace bumpy")
OUT.write_text("\n".join(out) + "\n")
print(f"wrote {OUT} ({WORLDS} worlds, {sum(len(w) for w in worlds)} nodes)")
```

using bumpy::world_node;

TEST_CASE("generated world-1 graph reproduces the historical kWorld1") {
    REQUIRE(world_node_count(1) == 15);
    // {up, down, left, right, x, y} for nodes 1..15 (was src/game/world_map.cpp kWorld1).
    const bumpy::MapNode want[15] = {
        {0, 0, 0, 2, 32, 32},    {0, 9, 1, 0, 112, 32},   {0, 0, 0, 4, 192, 32},
        {0, 7, 3, 0, 272, 32},   {0, 8, 0, 0, 32, 80},    {0, 10, 0, 7, 192, 80},
        {4, 0, 6, 0, 272, 80},   {5, 12, 0, 9, 32, 128},  {2, 0, 8, 0, 112, 128},
        {6, 0, 0, 11, 192, 128}, {0, 15, 10, 0, 272, 128}, {8, 0, 0, 13, 32, 176},
        {0, 0, 12, 14, 112, 176}, {0, 0, 13, 15, 192, 176}, {11, 0, 14, 0, 272, 176},
    };
    for (int node = 1; node <= 15; ++node) {
        const bumpy::MapNode& got = world_node(1, node);
        const bumpy::MapNode& w = want[node - 1];
        REQUIRE(got.up == w.up);
        REQUIRE(got.down == w.down);
        REQUIRE(got.left == w.left);
        REQUIRE(got.right == w.right);
        REQUIRE(got.x == w.x);
        REQUIRE(got.y == w.y);
    }
}

TEST_CASE("every world graph is sane, connected, and symmetric") {
    for (int world = 1; world <= bumpy::kWorldCount; ++world) {
        const int count = world_node_count(world);
        REQUIRE(count >= 1);
        REQUIRE(world_node(world, 0).right == 0);  // node 0 is the zero sentinel

        for (int n = 1; n <= count; ++n) {
            const bumpy::MapNode& node = world_node(world, n);
            REQUIRE(node.x >= 0);
            REQUIRE(node.x < 320);
            REQUIRE(node.y >= 0);
            REQUIRE(node.y < 200);
            for (int nb : {int(node.up), int(node.down), int(node.left), int(node.right)}) {
                REQUIRE(nb >= 0);
                REQUIRE(nb <= count);  // links reference valid nodes (0 = no link)
            }
        }

        // Symmetric links: my up <-> their down, my left <-> their right.
        for (int n = 1; n <= count; ++n) {
            const bumpy::MapNode& node = world_node(world, n);
            if (node.up != 0) REQUIRE(world_node(world, node.up).down == n);
            if (node.down != 0) REQUIRE(world_node(world, node.down).up == n);
            if (node.left != 0) REQUIRE(world_node(world, node.left).right == n);
            if (node.right != 0) REQUIRE(world_node(world, node.right).left == n);
        }

        // Every node reachable from node 1 (clearing a world requires clearing all nodes).
        std::set<int> seen{1};
        std::vector<int> stack{1};
        while (!stack.empty()) {
            const int cur = stack.back();
            stack.pop_back();
            const bumpy::MapNode& node = world_node(world, cur);
            for (int nb : {int(node.up), int(node.down), int(node.left), int(node.right)}) {
                if (nb != 0 && seen.insert(nb).second) stack.push_back(nb);
            }
        }
        REQUIRE(static_cast<int>(seen.size()) == count);
    }
}

namespace bumpy {

// All per-world *disk* resources: the D{n} level data and the decoded MONDE{n}.VEC
// backdrop (which also carries the world's map palette at offset 0x33). World-independent
// assets (the BUMSPJEU sprite bank, the DDFNT2 font) are loaded once by the caller, not
// here. The SDL shell owns one of these and reloads it when App requests a new world.
class WorldResources {
public:
    static WorldResources load(const std::filesystem::path& root, int world);

    [[nodiscard]] int world() const noexcept { return world_; }
    [[nodiscard]] const LevelResources& level() const noexcept { return level_; }
    // Decoded 320x200 screen-format MONDE{n}.VEC bytes (the span views owned storage).
    [[nodiscard]] std::span<const std::uint8_t> backdrop() const noexcept { return backdrop_; }
    // Number of completable boards = number of map nodes (the DEC tile-board count).
    [[nodiscard]] std::size_t board_count() const noexcept { return level_.board_count(); }

private:
    WorldResources(int world, LevelResources level, std::vector<std::uint8_t> backdrop)
        : world_(world), level_(std::move(level)), backdrop_(std::move(backdrop)) {}

    int world_{};
    LevelResources level_;
    std::vector<std::uint8_t> backdrop_;  // owned decoded MONDE{n}.VEC; backdrop() views it
};

}  // namespace bumpy

WorldResources WorldResources::load(const std::filesystem::path& root, int world) {
